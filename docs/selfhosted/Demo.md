# Try SlugBase Cloud

Try SlugBase without any setup! Sign up for a free account at SlugBase Cloud to explore all core features—bookmarks, folders, tags, and link forwarding.

## Access SlugBase Cloud

**URL:** [https://slugbase.app](https://slugbase.app)

1. Open [https://slugbase.app](https://slugbase.app)
2. Sign up with your email or log in with Google, Microsoft, or GitHub
3. Start using SlugBase immediately—no installation or configuration required

The **Free Plan** gives you a full-featured account to try SlugBase. When you're ready, you can upgrade to a paid plan for more capacity or self-host for complete control over your data.

## Features to Explore

1. **Bookmarks** – Add bookmarks with titles, URLs, and optional custom slugs
2. **Folders & Tags** – Organize with folders and flexible tagging
3. **Global Search** – Press `Ctrl+K` to search across all content
4. **View Modes** – Switch between card and list views
5. **Link Forwarding** – Enable short redirect URLs via `/go/:slug`
6. **Sharing** – Share bookmarks and folders with teams (on applicable plans)
7. **Themes** – Switch between light and dark mode in profile settings
