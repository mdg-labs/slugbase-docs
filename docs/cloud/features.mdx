# Features and User Guide

SlugBase Cloud uses the same app as self-hosted SlugBase. The following guides describe how to use bookmarks, folders, tags, sharing, and other features. **In SlugBase Cloud, all app routes are under `/app`** (e.g. `/app/bookmarks`, `/app/profile`, `/app/admin/billing`).

## Core features

- [Dashboard](/cloud/dashboard) – Overview of your bookmarks, stats, recent bookmarks, and quick navigation
- [Bookmarks](/cloud/bookmarks) – Managing your bookmark collection
- [Folders](/cloud/folders) – Organizing bookmarks with folders
- [Tags](/cloud/tags) – Tagging and filtering bookmarks
- [Shared](/cloud/shared) – Accessing and managing shared content

## Account and settings

- [Profile](/cloud/profile) – Account settings, preferences, theme, and (on Cloud) organization switcher
- [Login and auth](/cloud/auth) – How to sign in to SlugBase Cloud
- [Login page](/cloud/login) – Sign-in and sign-up entry points
- [Password reset](/cloud/password-reset) – Recovering your password
