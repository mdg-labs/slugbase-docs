## üîÑ Automatic Reset

The demo database **automatically resets every hour** to restore the default state. This ensures:

- ‚úÖ Fresh demo data for every visitor
- ‚úÖ Consistent demonstration experience
- ‚úÖ No data persistence between sessions
- ‚úÖ Admin can manually reset if needed

## ‚ú® Features to Try

### As Regular User (Alice or Bob)

1. **Explore Bookmarks** - Browse pre-configured bookmarks
2. **Create Bookmarks** - Add your own bookmarks with custom slugs
3. **Organize with Folders** - Create folders to group bookmarks
4. **Add Tags** - Tag bookmarks for flexible categorization
5. **Global Search** - Press `Ctrl+K` to search across all content
6. **View Modes** - Switch between card and list views
7. **Bulk Actions** - Select multiple bookmarks for batch operations
8. **Pinned Bookmarks** - Pin important bookmarks
9. **Shared Content** - Check the "Shared" page for bookmarks shared with you

### As Admin User

Everything from regular users, plus:

1. **Admin Panel** - Access to admin dashboard
2. **User Management** - View and manage all users
3. **Team Management** - Create and manage teams
4. **OIDC Provider Configuration** - Configure authentication providers
5. **System Settings** - Configure SMTP and other system settings
6. **Manual Demo Reset** - Reset the demo database manually (footer button)

## üéØ Demo Limitations

- **Data Resets Hourly** - All changes are lost on reset
- **No Email Functionality** - SMTP is disabled in demo mode
- **No OIDC Providers** - Only local authentication works
- **No Data Export** - Changes can't be exported (will be reset)
